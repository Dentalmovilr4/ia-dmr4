# 💎 Ecosistema Solana & DMR4 Coin

## Sobre Solana (SOL)
Solana utiliza un consenso único llamado **Proof of History (PoH)** combinado con Proof of Stake (PoS), lo que permite procesar hasta 65,000 transacciones por segundo con costos mínimos.

## Estrategia para DMR4 Coin
Para que el token "tenga vida" (liquidez y movimiento), se requiere:

### ⚙️ Inyección de Liquidez (Recarga)
1. **Añadir Liquidez:** Debes depositar un par (ejemplo: SOL/DMR4) en un DEX como **Raydium** o **Orca**. 
2. **Burn de LP:** Para dar confianza, muchos creadores "queman" los tokens de liquidez para asegurar que no habrá un "Rug Pull".
3. **Market Making:** Realizar pequeñas transacciones constantes para que los gráficos de *DexScreener* muestren actividad real.

### 🚀 Pasos para "Recargar" Vida al Token
- **Solana Faucet:** Si estás en Testnet, usa un faucet para obtener SOL de prueba.
- **Mainnet:** Necesitas enviar SOL real a la wallet creadora (la que tiene el Authority) para pagar los "rent fees" de las cuentas asociadas al token.
